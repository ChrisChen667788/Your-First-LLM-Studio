# v0.8.1 Adapter Lifecycle Evidence

Date: 2026-07-07

## Scope

This evidence promotes Adapter Lifecycle from a release-matrix placeholder into a feature-owned Fine-tune read-model and action surface.

## Implementation

- Added `AgentFineTuneAdapterLifecycleSummary` to the Fine-tune summary contract.
- Added `lib/finetune/lifecycle-service.ts` for:
  - adapter variant registry records,
  - variant diff evidence,
  - completed and planned export plans,
  - rollback proof lifecycle actions.
- Added `/api/finetune` actions:
  - `record-lifecycle-export-plan`
  - `run-lifecycle-rollback-proof`
- Added Fine-tune Assets UI for lifecycle registry totals.
- Added Fine-tune Assets lifecycle filters for status, diff conclusion, and export format.
- Added a variant detail drawer showing lineage, best-checkpoint evidence, diff deltas, export plans, and rollback proof count.
- Added per-adapter actions for merged/q8 export planning and rollback proof rehearsal.
- Updated the release evidence matrix to score v0.8.1 from lifecycle registry totals instead of hard-coded blockers.
