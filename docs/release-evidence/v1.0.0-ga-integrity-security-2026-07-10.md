# v1.0.0 GA Integrity And Security Evidence

Date: 2026-07-10

## Scope

This batch hardens non-cloud GA evidence without treating local checks as cloud production sign-off.

- Provider Ops snapshots now carry SHA-256 digests and expose explicit verified, missing, and invalid counts.
- The GA bundle carries one digest per source plus a bundle-level digest, verified history, JSON/Markdown export, and retention that preserves the latest archive.
- Route smoke history is available through a feature-owned API and an Experiments trend panel.
- Admin compatibility deletion sign-off binds a trusted operator identity to the current manifest digest and remains unavailable until all sunset gates are ready.
- Release security preflight scans tracked and untracked non-ignored candidate files without emitting secret values and records a production dependency audit summary.
- Admin Provider target cards and Agent runtime/connection shell state moved behind feature-owned rendering and composition ports.
