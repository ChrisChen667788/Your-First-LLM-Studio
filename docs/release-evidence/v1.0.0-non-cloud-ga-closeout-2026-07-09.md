# v1.0.0 Non-cloud GA Closeout Evidence

Date: 2026-07-09

## Scope

This evidence advances the GA closeout items that do not require real production cloud identity, KMS, or immutable object storage.

Cloud production sign-off remains separate and must still be generated with real workload identity and `requireCloud=true`.

## Added Evidence Contracts

- Added `admin.compatibility-sunset.v1` as an Admin-owned read-model for deprecated Admin API wrappers.
- The sunset read-model separates:
  - route-smoke hits,
  - runtime legacy hits,
  - expected wrapper routes and methods,
  - required header-smoke coverage,
  - deletion readiness and blockers.
- Added `first-llm-studio.route-smoke.v1` JSON output from `npm run smoke:routes` at `output/release-smoke/route-smoke-latest.json`.
- Added `experiments.route-smoke-evidence.v1` as a release-matrix input over the latest route-smoke artifact.
- Migrated Fine-tune report preview and bundle download links from the deprecated `/api/admin/finetune` wrapper to the canonical `/api/finetune` route.
- Compatibility usage evidence now separates source-tagged runtime hits, route-smoke hits, and historical unclassified hits recorded before source tagging was available.
- Added `POST /api/admin/compatibility-usage` action `archive-historical-unclassified`, which writes a durable historical archive and clears only the legacy-unclassified bucket when source-tagged runtime hits are zero.

## Product Surface

- `/api/admin/compatibility-usage` now returns both usage summary and sunset evidence.
- `/api/admin/compatibility-usage` can archive and clear historical unclassified evidence without deleting current route-smoke proof.
- `/api/admin/dashboard` now includes `adminCompatibilitySunset`.
- `/admin` renders the compatibility sunset evidence through a dedicated `AdminCompatibilitySunsetPanel`, keeping the main Admin dashboard thinner.

## GA Matrix Integration

`v1.0.0` release evidence now includes:

- promotion gate status,
- docs/screenshot freshness,
- public release demo capture,
- cross-surface route smoke artifact,
- Admin compatibility sunset evidence,
- benchmark evidence,
- provider request evidence,
- adapter export evidence.

The GA matrix is expected to remain non-final while:

- real cloud v0.9 sign-off is pending,
- runtime consumers still hit deprecated Admin APIs,
- the 2026-09-30 compatibility sunset has not passed.

As of the 2026-07-09 verification run, historical unclassified compatibility
hits were archived and cleared from the live usage summary. Future route smoke
coverage is retained as source-tagged smoke evidence and should not re-open the
historical compatibility blocker.

## Verification

Run locally:

```bash
npm run typecheck:changed
SMOKE_CURL_MAX_TIME=90 npm run smoke:routes
```

Then inspect:

```bash
curl -fsS http://localhost:3011/api/admin/compatibility-usage
curl -fsS http://localhost:3011/api/experiments/release-evidence-matrix
```

Archive historical compatibility evidence:

```bash
curl -fsS http://localhost:3011/api/admin/compatibility-usage \
  -H "Content-Type: application/json" \
  -d '{"action":"archive-historical-unclassified","clear":true}'
```
