## Scope

This evidence promotes the Deployment track from a read-only `/api/deployment` stub into a durable local production-control rehearsal.

## Implementation

- Added `features/deployment/contracts.ts` with `deployment.control-plane.v1`.
- Added `features/deployment/control-plane.ts` for:
  - durable usage outbox JSONL records,
  - external audit archive evidence files,
  - KMS-style Ed25519 receipt signing and verification,
  - failover rehearsal records with RPO/RTO counters.
- Updated `/api/deployment`:
  - `GET` returns the control-plane read-model,
  - `POST` accepts `action=rehearse-production-control-plane`,
  - unsupported registry mutations still fail closed with `409`.
- Updated `experiments.release-evidence-matrix.v1` so `v0.9.0` is scored from real control-plane evidence instead of hard-coded placeholders.
- Added route smoke coverage for the deployment control-plane contract.
